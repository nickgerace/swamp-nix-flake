# swamp-nix-flake

![GitHub License](https://img.shields.io/github/license/nickgerace/swamp-nix-flake)

A [swamp.club](https://swamp.club) extension for [nix flakes](https://wiki.nixos.org/wiki/Flakes).

## Developing

Install [`just`](https://github.com/casey/just) and run `just --list` to get started.
All targets map to `nix flake <command>` invocations.
